автоматизатор военкомата

приложения для любого, но одного отделения военкомата.
для персонала, занимающегося основными военкоматовскими делами (всех, кроме уборщиков и охранников).


нужны окна:
	вход (главное)
	выбор профиля
	администратор
	под каждого врача (стиль и наполнение одно, но у каждого есть свои приколы)
	дежурный
	архивщик
	тот, кто приниает решения по категориям



роли (по факту)(они все гниды):
	админ — решатель всех проблем, ставит базу, чинит базу, техштучки

	регистрирующий — заводит личные дела, выдаёт повестки, принимает людей, принимает документы 

	врач — проводит медосмотр, делает записи в личное дело

	комиссия — принимает решения о выдаче категории

	архивариус — роется в архиве, убирает документы в архив, ищет документы в архиве, потерять документы, подпереть стол


роли (в приложении):
	админ — придумаем потом

	регистрирующий:
		создать личное дело
		сохранить личное дело
		создать повестку 
		сохранить повестку
		выдать повестку

	врач:
		внести данные в чеклист
		добавить записи
		сохранить запись приёма

	комисиия:
		просмотреть личное дело
		внести записи
		сделать заключение о категории

	архивариус:
		найти документ по номеру
		найти документ по типу
		найти документ по человеку
		добавить документ
		распечатать документ
